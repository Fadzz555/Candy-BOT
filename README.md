# Cara Penginstalan

* Termux
```bash
---------------
> apt update
> apt upgrade
 Untuk Android 7 Kebawah
---------------
> pkg update
> pkg upgrade
 Untuk Android 7 Keatas
---------------
> git clone https://github.com/ZeroChanBot/Candy-BOT
> cd Candy-BOT
> bash install.sh
 Tunggu Hingga Proses Download Selesai
```

* Run Bot
```bash
> npm start
 Scan QR Code
```
